安装vim插件管理器vundle

Posted on 2013-05-27 Edited on 2020-12-04 In Vim

vundle可以自动管理vim插件，从自动下载安装到更新

安装
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

配置
编辑~/.vimrc

安装插件
打开vim,输入:BundleInstall即自动安装配置的插件，插件全部安装到~/.vim/bundle目录下
